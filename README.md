# Market Metrics Dashboard

Interactive Streamlit dashboard for analysing multi-asset performance using index data and live fund stitching.

## Features
- Multi-asset returns across time horizons
- Index + live fund stitching
- Absolute and relative performance views
- Inflation integration
- Full diagnostics transparency

## Tech
- Python
- Streamlit
- Pandas / NumPy
- yfinance

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
